
LibHaru/SML
===============================================================

.. contents::


StandardML(SML) [#smlnj]_ binding library to libHaru. [#libharu]_



Requirements
---------------------------------------------------------------

This binding library is developped with...

- libHaru 2.4.0
- SML/NJ 110.77


Environment
---------------------------------------------------------------

Linux/x86 is only supported platform.
But, code depends on Linux and/or x86 is not contained.
